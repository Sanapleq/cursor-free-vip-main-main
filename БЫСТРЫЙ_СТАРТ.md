### **Способ 2: Ручная установка**

```powershell
# 1. Откройте PowerShell или Command Prompt
cd "h:\cursor-free-vip-main-main"

# 2. Создайте виртуальное окружение
python -m venv myenv

# 3. Активируйте
.\myenv\Scripts\activate

# 4. Установите зависимости
pip install -r requirements.txt

# 5. Запустите
python main.py
```

## 🔧 **Возможные проблемы:**

### **❌ SETUP.bat не запускается**

**Решение:**
1. Убедитесь, что установлен Python 3.11-3.14
2. Проверьте: `python --version`
3. Переустановите Python с https://www.python.org/

### **❌ START.bat закрывается сразу**

**Решение:**
1. Запустите SETUP.bat сначала
2. Проверьте, что myenv/ существует
3. Запустите от имени администратора

### **❌ "Python не найден"**

**Решение:**
```
1. Установите Python: https://www.python.org/downloads/
2. При установке отметьте: "Add Python to PATH"
3. Перезапустите компьютер
4. Запустите SETUP.bat снова
```

### **❌ "Требуется запуск от администратора"**

**Решение:**
1. Кликните правой кнопкой на START.bat
2. Выберите "Запуск от имени администратора"

### **❌ Ничего не происходит при запуске**

**Решение - ручной запуск:**
```powershell
cd "h:\cursor-free-vip-main-main"
.\myenv\Scripts\activate
python main.py
```
