## 🔧 **Возможные проблемы:**

### **❌ "Python not found"**

**Решение:**
1. Установите Python 3.11-3.14
2. Скачайте с: https://www.python.org/downloads/
3. При установке отметьте: **"Add Python to PATH"**
4. Перезапустите компьютер
5. Запустите SETUP.bat снова

### **❌ "Virtual environment not found"**

**Решение:**
```
Запустите: SETUP.bat
```

### **❌ "Administrator rights required"**

**Решение:**
1. Кликните правой кнопкой на START.bat
2. Выберите "Запуск от имени администратора"

### **❌ Ничего не происходит**

**Решение - ручной запуск:**
```powershell
cd "h:\cursor-free-vip-main-main"
.\myenv\Scripts\activate
python main.py
```
