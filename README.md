# Cursor Free VIP

Автоматическая регистрация аккаунтов Cursor с поддержкой Google и GitHub OAuth.

## 🚀 Быстрый старт

### Один клик - всё работает!

**Просто запустите:** `RUN.bat`

Что делает:
- ✅ Проверит Python
- ✅ Создаст виртуальное окружение (если нет)
- ✅ Установит зависимости (1-3 минуты)
- ✅ Закроет Cursor
- ✅ Запустит программу

**Время:** Первый запуск 1-3 мин, следующие - 5-10 секунд

---

## ⚠️ Важно

**Требуются права администратора!**

При запуске программа запросит права администратора. Это необходимо для:
- Сброса Machine ID
- Работы с файлами Cursor
- Изменения системных настроек

---

## 📋 Системные требования

- **ОС:** Windows 10/11
- **Python:** 3.11-3.14
- **Права:** Администратор (обязательно)
- **Cursor:** Должен быть установлен

---

## 🔧 Если что-то не так

### "Python not found"
```
Установите Python: https://www.python.org/downloads/
При установке отметьте: "Add Python to PATH"
```

### "Administrator rights required"
```
Запустите RUN.bat от имени администратора:
ПКМ на RUN.bat → "Запуск от имени администратора"
```

### "Dependencies not installed"
```
Удалите папку myenv/ и запустите RUN.bat снова
```
