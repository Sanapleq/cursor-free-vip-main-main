# Cursor Free VIP

Автоматическая регистрация аккаунтов Cursor с поддержкой Google и GitHub OAuth.

## Системные требования

- **ОС:** Windows 10/11
- **Python:** 3.11-3.14
- **Права:** Администратор (обязательно)
- **Cursor:** Должен быть установлен

## Быстрый старт

### 1. Установка зависимостей

Дважды кликните на **SETUP.bat**

- Автоматически создаст виртуальное окружение
- Установит все 29 необходимых пакетов
- Создаст ярлык на рабочем столе

**Время установки:** 3-7 минут

### 2. Запуск программы

Дважды кликните на **START.bat**

- Проверит права администратора
- Активирует виртуальное окружение
- Запустит программу

**Время запуска:** 5-10 секунд

## Важные замечания

⚠️ **Требуются права администратора** - программа не будет работать без них

⚠️ **Закройте Cursor перед запуском** - все процессы Cursor должны быть закрыты

⚠️ **Только для Windows** - другие ОС не поддерживаются

⚠️ **Образовательные цели** - используйте на свой страх и риск

## Решение проблем

### "Python not found"
Установите Python 3.11-3.14 с https://www.python.org/downloads/
При установке отметьте "Add Python to PATH"

### "Virtual environment not found"
Запустите SETUP.bat ещё раз

### "Administrator rights required"
Запустите START.bat от имени администратора

### "ModuleNotFoundError"
Запустите SETUP.bat для установки зависимостей

### Файл заблокирован
Закройте Cursor и другие программы, затем запустите снова

## Поддержка

Если возникли проблемы:
1. Убедитесь что Python установлен
2. Запустите SETUP.bat
3. Запустите START.bat от имени администратора
4. Убедитесь что Cursor закрыт
