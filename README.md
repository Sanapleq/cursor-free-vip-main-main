# Cursor Free VIP

## 🚀 Запуск

```powershell
.\RUN.bat
```

**ВСЁ!** Больше ничего делать не нужно.

---

## ⚡ Что делает RUN.bat

- ✅ Проверяет Python
- ✅ Создаёт виртуальное окружение (если нет)
- ✅ Устанавливает зависимости (если нет)
- ✅ Закрывает Cursor
- ✅ Запускает программу

**Время:** Первый запуск 1-3 мин, следующие - 5 сек

---

## 🛠️ Требования

- Windows 10/11
- Python 3.11-3.14
- Права администратора

---

## 🔧 Если не работает

### "Python not found"
```
Установите Python: https://www.python.org/
Отметьте: "Add Python to PATH"
Перезапустите компьютер
```

### "Administrator rights required"
```
ПКМ на RUN.bat → "Запуск от имени администратора"
```

### Всё остальное
```
Удалите папку myenv/ и запустите RUN.bat снова
```
