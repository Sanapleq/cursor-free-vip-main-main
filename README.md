# Cursor Free VIP

Автоматическая регистрация аккаунтов Cursor с поддержкой Google и GitHub OAuth.

![Version](https://img.shields.io/badge/version-1.11.06-blue)
![Platform](https://img.shields.io/badge/platform-Windows-green)
![Python](https://img.shields.io/badge/python-3.11--3.14-blue)

## ⚡ Быстрый старт

### Один запуск - всё работает!

**Просто запустите:** `QUICK_START.bat`

Что делает:
- ✅ Проверит Python
- ✅ Создаст виртуальное окружение
- ✅ Установит зависимости (1-3 минуты)
- ✅ Запустит программу

**Время:** Первый запуск 1-3 мин, следующие - 5-10 секунд

---

## 📋 Системные требования

- **ОС:** Windows 10/11
- **Python:** 3.11-3.14
- **Права:** Администратор (обязательно)
- **Cursor:** Должен быть установлен

## 🚀 Установка и запуск

### Способ 1: Быстрый старт (РЕКОМЕНДУЕТСЯ)

```batch
# Просто запустите
QUICK_START.bat
```

**Что происходит:**
1. Проверяет Python
2. Создаёт `myenv/` (если нет)
3. Устанавливает зависимости (оптимизировано, 1-3 мин)
4. Запускает программу

### Способ 2: Пошагово

```batch
# 1. Установка (один раз)
SETUP.bat

# 2. Запуск (каждый раз)
START.bat
```

## 🔧 Решение проблем

### "Python not found"
```
Установите Python 3.11-3.14:
https://www.python.org/downloads/

При установке отметьте: "Add Python to PATH"
```

### "Virtual environment not found"
```
Запустите SETUP.bat
```

### "Administrator rights required"
```
Запустите от имени администратора:
ПКМ на файле → "Запуск от имени администратора"
```

### "ModuleNotFoundError"
```
Запустите SETUP.bat для установки зависимостей
```

### Файл заблокирован
```
Закройте Cursor и другие программы
Затем запустите снова
```
