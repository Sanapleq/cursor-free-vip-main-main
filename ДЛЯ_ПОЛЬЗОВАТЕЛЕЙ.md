## ⚠️ Важные моменты:

### **Требуются права администратора!**

Программа автоматически запросит права при запуске. Это необходимо для:
- Изменения файлов Cursor
- Сброса Machine ID
- Работы с реестром Windows

### **Если нет прав администратора:**

1. Кликните **правой кнопкой** на START.bat
2. Выберите **"Запуск от имени администратора"**

## 🔧 Возможные проблемы:

### **❌ "Python не найден"**

**Решение:**
1. Установите Python 3.11-3.14
2. Скачайте с: https://www.python.org/downloads/
3. При установке отметьте: **"Add Python to PATH"**

### **❌ "Виртуальное окружение не найдено"**

**Решение:**
```
Запустите: SETUP.bat
```

### **❌ "Требуется запуск от администратора"**

**Решение:**
1. Кликните правой кнопкой на START.bat
2. Выберите "Запуск от имени администратора"

### **❌ Программа не открывается**

**Решение:**
```
1. Откройте PowerShell
2. Перейдите в папку: cd "h:\cursor-free-vip-main-main"
3. Запустите: .\myenv\Scripts\activate
4. Запустите: python main.py
```
