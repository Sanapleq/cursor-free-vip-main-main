# Cursor Obxod - Простой обход Cursor

## 🚀 Быстрый старт

```batch
QUICK_START.bat
```

**ВСЁ!** Программа сама всё сделает.

---

## ⚡ Что внутри

- ✅ Автоматическая установка
- ✅ Закрытие Cursor перед запуском  
- ✅ Сброс Machine ID
- ✅ Регистрация аккаунтов (Google/GitHub)
- ✅ Обход лимитов токенов
- ✅ Отключение обновлений

**Время:** Первый запуск 1-3 мин, следующие - 5 сек

---

## 🛠️ Требования

- Windows 10/11
- Python 3.11-3.14
- Права администратора

---

## 🔧 Если что-то не так

### "Python not found"
Установите Python: https://www.python.org/
Отметьте: "Add Python to PATH"

### "Administrator rights required"  
ПКМ на файле → "Запуск от имени администратора"

### Всё остальное
Запустите `SETUP.bat` ещё раз
